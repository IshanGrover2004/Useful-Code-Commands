
# Neovim

Installation (AstroVim)
~ git clone --depth 1 https://github.com/AstroNvim/AstroNvim ~/.config/nvim

Uninstall -:
~ sudo apt-get remove --auto-remove neovim
~ rm -rf .config/nvim && rm -rf.local/share/nvim
